Ajuda e Suporte

Informações sobre como usar o sistema

Como usar o sistema:

  1. Selecione o tipo de operação (Bloqueio ou Desbloqueio)
  2. Para bloqueio, informe a data no formato DD/MM/AAAA
  3. Cole os CPFs/Matrículas no campo esquerdo (um por linha)
  4. Cole os nomes no campo direito (mesma ordem dos CPFs)
  5. Clique em "Processar Dados"
  6. Use os botões para baixar, copiar ou imprimir o resultado
